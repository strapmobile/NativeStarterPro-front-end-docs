Installation

Installation procedure

1. Downloading the code

Option #1: Download zip here

Option #2: Clone the repo with git clone http://gitstrap.com/strapmobile/NativeStarterPro-with-backend-integration.git

Option #3: Follow this guide to clone using GitStrap which will allow easier syncing with future updates.

2. Running the app

$ cd NativeStarterPro-with-backend-integration
$ npm install
$ react-native link

For iOS:

$ react-native run-ios

For Android:

$ react-native run-android
